## Tech Challenge – Fase 5
# Sistema Preditivo e Painel Analítico de risco de defasagem dos alunos da Instituição Passos Mágicos

Este projeto tem como objetivo **analisar e prever o risco de um aluno entrar em defasagem escolar**, combinando modelagem preditiva com IA e análise exploratória de dados por meio de um painel analítico interativo.  
A solução foi desenvolvida para apoiar análises educacionais, propostas de melhorias e tomada de decisão baseada em dados.


## Objetivos do Projeto

- Desenvolver um modelo de machine learning capaz de predizer o risco de um aluno entrar em defasagem em relação à fase de ensino ideal para sua idade
- Criar um aplicativo interativo para predição individual
- Construir um painel analítico com insights relevantes sobre o desempenho atual dos alunos, indicadores importantes e seu impacto para análises e identificar possibilidades de melhoria no processo educacional atual


## Funcionalidades

### Preditor de Risco de Defasagem (Streamlit)
- Interface interativa e intuitiva
- Entrada de dados educacionais do indivíduo
- Cálculo automático do risco de defasagem
- Painel com os resultados do usuário
- Análise dos indicadores
- Aplicação totalmente em português


### Painel Analítico (Streamlit)
- Distribuição por níveis de risco (sem risco, atenção, risco moderado e risco alto)
- Comparação de indicadores e seu impacto (IAA, IEG, IPS, IDA e INDE)
- Taxa de risco e volume por ano, gênero e categoria de instituição


## Tecnologias Utilizadas

- Python
- Pandas / NumPy
- Scikit-learn
- Streamlit
- Git & GitHub
